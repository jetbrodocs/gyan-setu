# Questions for Management Team

Items flagged during demo scoping that need management input. Until resolved, the noted default decision stands.

---

### 1. Multi-language support in demo
**Context:** The PRD specifies 4 languages (English, Gujarati, Hindi, Sanskrit). The language selector will be visible in the UI.

**Question:** Should the demo actually support switching UI language, or is English-only acceptable for the demo day?

**Current default:** English only. Language selector shown but non-functional.

**Impact if yes:** Requires translating all UI strings into 3 additional languages, plus ensuring RTL/font support for Gujarati and Hindi. Significant effort increase.

---

### 2. Video Library — YouTube embeds or self-hosted?
**Context:** Video Library module needs seeded video content from teachers/creators. YouTube embeds are the quickest path.

**Question:** Are YouTube embeds acceptable for the demo, or should videos be self-hosted from a public dataset?

**Current default:** YouTube embeds.

**Impact if no:** Need to source and host a public video dataset, which adds storage and streaming complexity.
