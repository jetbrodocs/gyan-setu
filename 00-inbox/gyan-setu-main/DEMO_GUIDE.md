## What to Test

### 1. Dashboard
- Welcome banner, stats cards, "Continue Reading" section
- Trending books grid with colored covers
- Sidebar navigation to all modules
- Collapsible sidebar (burger menu icon)

### 2. Digital Library
- Browse 27 books across 4 languages (Gujarati, Hindi, English, Sanskrit)
- Filter by Language, Format, Collection using the sidebar
- Sort by Rating or Year
- Click any book → Book Detail page with AI summary

### 3. eBook Reader
- From a book detail page, click "Read Now" (available for 12 books)
- Table of Contents on the left — click chapters to navigate
- Font size controls (+ / -)
- AI Reading Companion on the right — ask it questions about what you're reading
  - Try: "What is this chapter about?", "Translate this to Hindi", "Explain the main themes"

### 4. Audiobooks
- Browse 9 audiobooks (Tagore, Gandhi, Bhagavad Gita, etc.)
- Click any audiobook → Player with play/pause, skip, speed control (0.75x–2x)
- Chapter list on the right
- Audio streams from archive.org (LibriVox public domain)

### 5. Newspapers & Periodicals
- Browse 10 newspaper editions (Gujarat Samachar, Sandesh, Times of India, etc.)
- Filter by Language, City, Date
- Mini calendar shows available dates
- Click a newspaper → full-page viewer

### 6. Test Prep Hub
- 6 exam categories (UPSC, GPSC, SSC, Banking, GATE, JEE/NEET)
- Question of the Day — try answering it
- My Performance section with readiness stats and subject-wise progress
- Click a category → see available tests → click "Start Test"

### 7. Mock Test Interface
- Full-screen exam mode with countdown timer
- Navigate between questions using the grid on the right
- Mark questions for review
- Submit and see detailed results with score breakdown

### 8. Podcast Studio (login as teacher@demo.com)
- Left panel: Record audio using your microphone
- Fill in episode title, select a series
- Publish → episode appears in the Podcasts consumer page
- Right panel: My Podcasts with series list and stats

### 9. Podcasts (Consumer)
- Browse published podcast episodes across 3 series
- Filter by series
- Play episodes with built-in audio player

### 10. IKS Heritage
- Historical timeline: Vedic → Mauryan → Gupta → Solanki → Mughal → Modern
- Click an era to see manuscripts from that period
- Filter by language
- 360° Virtual Tour cards (coming soon placeholders)

### 11. STEM Lab
- Category tabs: Coding, Robotics, AI & ML, Electronics, 3D Design, Atal Innovation
- Video lessons with YouTube player
- Code editor with Run button
- Badges, leaderboard, and live workshops sidebar

### 12. User Profile (My Analytics)
- User info with membership tier and streak
- Stats cards: Books Read, Audiobooks, Tests, Podcast Hours
- 30-day reading activity bar chart
- Currently Reading with progress bars
- Earned Certificates, My Bookshelf, Recent Test History

### 13. Membership Plans
- Three tiers: BASIC (Free), STANDARD (₹99/mo), GOLD (₹199/mo)
- Feature comparison with checkmarks
- Current plan highlighted
- 50% student/senior discount note

## Known Limitations
- Book/audiobook covers are placeholder SVGs (not real cover art)
- Newspaper pages are sample layouts (not real editions)
- Code editor in STEM Lab is a demo textarea (no real Python execution)
- 360° virtual tours are placeholder buttons
- App may take ~5 seconds to wake up on first visit (scale-to-zero hosting)
- AI Reading Companion requires API key to be configured
