# Module 13: Video Library

**Reference Screenshot**: None (new module, not in original presentation)

## Description
Browse and watch educational videos created by teachers and content creators. Similar to Digital Library but for video content. Consumer-side only for the demo.

## Data Source
- YouTube embeds (pending management approval) OR public video dataset
- Metadata: title, teacher/creator name, subject, category, duration

## User Flow
1. Video Library → Browse/search/filter videos
2. Select video → Watch in simple embed player

## Specifications

### Browse Page
- Grid/list of video cards (similar layout to Digital Library)
- **Search**: Functional across seeded video content
- **Filters**: By category/subject (driven by dataset)
- Video card: Thumbnail, title, creator name, duration, category tag

### Video Player
- Simple embed player (YouTube embed or standard HTML5 video)
- No custom features (no speed control, notes, etc.)
- Standard browser video controls

### What's NOT Included
- No upload flow for creators (seeded content only)
- No comments or social features
- No custom player controls
